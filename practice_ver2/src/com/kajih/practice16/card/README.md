## トランプゲーム  

### 説明  
トランプゲームで利用できるカードのクラスである  
カードはマーク、数字が決められており、ジョーカーはない  

|プロパティ|説明|
|---|---|
|マーク|ダイヤ、スペード、ハート、クローバー|
|数字|1〜13、1はA、11はJ、12はQ、13はK|

### クラス  
+ カード(Card)  
マーク、数字を保持する

+ カードマネジャー(CardMng)  
それぞれのマークを持つ1〜13のカードを生成する  
カードは合計52枚  
また、以下のメソッドを持つ

|メソッド|説明|
|---|---|
|reset|カードを52枚にし、初期の並びに戻す|
|shuffle|残りのトランプをシャフル|
|draw|カードを一枚引く|
